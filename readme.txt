WP 2 Step Facebook Authentication provides an extra layer of facebook authentication to wordpress account.

== Description ==
WP 2 Step Facebook Authentication is a light weight stable plugin that provides an extra security layer of facebook authentication to wordpress account.
It comes with minimal configuration and works with latest version of Wordpress.

== Installation ==
1. Upload "wp2step-fb-authentication" folder to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
1. Provide Facebook AppID and App secret key at the plugin's backend section.
1. Done.

== Frequently Asked Questions ==
= Does it requires any configuration? =
You need to configure Facebook AppID and App secret key, thats it.

= Does it works with all Wordpress website? =
Yes.
